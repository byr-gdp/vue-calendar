# vue-calendar

## Param

* value: 缺省值是今天，除非你特地表明 "null"。可填入具体日期形式如 "2015-12-18" ，也可填入 "today" ，自动帮你处理。
* begin: 开始时间，缺省无限制。可填入具体日期形式如"2015-12-18"，也可填入"1y"、"1m"、"1d"表示相对于当前日期往前推。
* end: 结束时间，缺省无限制。可填入具体日期形式如"2015-12-18"，也可填入"1y"、"1m"、"1d"表示相对于当前日期往前推。
